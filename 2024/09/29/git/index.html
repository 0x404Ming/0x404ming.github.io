    以下是一些常用的 Git 工具和命令：

    常用 Git 命令

    1. 配置 Git：

      git config --global user.name "你的名字"
git config --global user.email "你的邮箱"

    2. 初始化仓库：

      git init

    3. 克隆仓库：

      git clone <仓库地址>

    4. 查看状态：

      git status

    5. 添加更改：

      git add <文件名>
git add .  # 添加所有更改

    6. 提交更改：

      git commit -m "提交信息"

    7. 查看提交历史：

      git log

    8. 分支管理：

      • 创建新分支：

        git branch <分支名>

      • 切换分支：

        git checkout <分支名>

      • 合并分支：

        git merge <分支名>

    9. 推送和拉取：

      • 推送更改：

        git push origin <分支名>

      • 拉取更新：

        git pull

    以下是 Git 中常用的删除命令和操作：

    删除文件

    1. 从暂存区和工作区删除文件：

      git rm <文件名>

    2. 只从工作区删除文件，但保留在暂存区：

      git rm --cached <文件名>

    删除分支

    1. 删除本地分支：

      git branch -d <分支名>  # 删除已合并的分支
git branch -D <分支名>  # 强制删除未合并的分支

    2. 删除远程分支：

      git push origin --delete <分支名>

    删除标签

    1. 删除本地标签：

      git tag -d <标签名>

    2. 删除远程标签：

      git push origin --delete <标签名>

    其他删除操作

    • 撤销上次提交，但保留更改：

      git reset --soft HEAD~1

    • 撤销上次提交和更改：

      git reset --hard HEAD~1

    其他工具

    • 图形界面工具：如 GitKraken、SourceTree 和 GitHub Desktop，适合不喜欢命令行的用户。
    • 命令行工具：如 tig 和 gitk，用于可视化 Git 仓库状态和历史。
